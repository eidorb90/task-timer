# Task Timer

A simple and efficient task timing application.

## Features
- Track time spent on tasks
- Start, pause, and stop timers
- View task history
- Export time logs

## Installation
```bash
npm install
npm start
```

## Usage
1. Create a new task
2. Start the timer
3. Pause or stop as needed
4. View your time logs
